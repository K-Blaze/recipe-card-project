# Recipe Card React App

## Description
This project is a React application that displays multiple recipe cards using reusable components. Each card shows the recipe name, cook time, image, and ingredients in a clean and styled layout.

## Features
- Uses React functional components
- Displays dynamic data using `.map()`
- Passes data using props
- Organized file structure
- Styled with CSS for a modern card layout

## Technologies Used
- React
- JavaScript
- CSS
